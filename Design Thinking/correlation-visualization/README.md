# 📊 Correlation Visualization using Python

## 📌 Overview

This project analyzes the relationship between different academic subjects using a correlation matrix and visualizes the results using a heatmap.

Understanding correlations helps identify how strongly variables are related to each other.

---

## 📂 Dataset

The dataset contains marks of students in:

* Maths
* Physics
* Chemistry
* English

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib

---

## 🔍 Features

* Data cleaning (handling duplicates)
* Correlation matrix calculation
* Heatmap visualization
* Basic interpretation of results

---

## 📊 Output

* Displays correlation values between subjects
* Visual heatmap showing relationships

---

## 📖 Key Insights

* Subjects like Maths and Physics show strong positive correlation
* English shows relatively weaker correlation with science subjects

---

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run the notebook
